How This Works

The Caesar Cipher shifts each alphabet letter by a fixed number. For example, with shift 3, A -> D and x -> a. Decryption reverses the shift, restoring the original message.

ROT13 is a special Caesar Cipher with a fixed shift of 13, which means encrypting twice returns the original text.
